## Build

Clone:
```
$ git clone git@github.com:dylanrainwater/sample_app.git
```

Install gems:
```
$ bundle install --without production
```

Migrate database:
```
$ rails db:migrate
```

Run test suite
```
$ rails test
```

Finally, launch a local server!
```
$ rails server
```

## Built with

* Ruby 2.3.3

* Rails 5.0.2
